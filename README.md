## Adding materials each week

After each class, add up to three files and update `index.html`:

| Step | File to add | Naming pattern |
|------|-------------|----------------|
| 1 | Lecture slides (PDF) | `slides/lecture-04-slides.pdf` |
| 2 | Lecture summary | `lectures/lecture-04-summary.html` |
| 3 | Breakout solutions | `solutions/lecture-04-solutions.html` |

Then open `index.html`, find that lecture's entry in the `lectures` array,
and replace `null` with the file path for each item you added. GitHub Pages
redeploys automatically within about 60 seconds of pushing.

**Canvas:** After updating GitHub, also open `canvas-home.html` in a text
editor, copy its full contents, and paste into your Canvas home page's HTML
editor (`</>` button). Update the lecture row's background colour and add
the three link buttons (copy from the Lecture 03 row).
