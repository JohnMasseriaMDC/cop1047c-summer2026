Your Tasks

  1. Write a while loop that prints all multiples of 3 from 3 to 30. Then rewrite it using for and range().

# ----------------------------------------------------------
# PROBLEM 1: Multiples of 3 from 3 to 30
# Write it two ways.
# ----------------------------------------------------------

print("--- Problem 1: Multiples of 3 ---")

# While loop version
num = 3
while num <= 30:
    print(num, end=" ")
    num += 3
print()   # newline after the row

# For loop + range() version — cleaner!
for num in range(3, 31, 3):   # start=3, stop=31 (not included), step=3
    print(num, end=" ")
print()

# Sample output (both produce the same result):
# 3 6 9 12 15 18 21 24 27 30

# PROBLEM 3: Enter 5 numbers, print total and average
# ----------------------------------------------------------

print("\n--- Problem 3: Running total ---")

total = 0

for i in range(5):
    num = float(input(f"Enter number {i + 1}: "))
    total += num

average = total / 5
print(f"Total:   {total:.2f}")
print(f"Average: {average:.2f}")

# ----------------------------------------------------------
# CHALLENGE: Sentinel value — keep accepting numbers
#            until the user enters -1
# ----------------------------------------------------------

print("\n--- Challenge: Sentinel loop ---")

count = 0
total = 0
num = float(input("Enter a number (-1 to stop): "))

while num != -1:
    total += num
    count += 1
    num = float(input("Enter a number (-1 to stop): "))

if count > 0:
    print(f"Numbers entered: {count}")
    print(f"Average: {total / count:.2f}")
else:
    print("No numbers entered.")

# Sample interaction (input: 10, 20, 30, -1):
# Numbers entered: 3
# Average: 20.00

Your Tasks

  1. Use nested for loops to print a seating chart. Rows are numbered; columns use letters A, B, C… Use chr(ord("A") + col) to get letters from a column index.
  2. Build a sum program in three versions: Version 1 — just print(i) to verify the loop runs. Version 2 — add an accumulator to sum 1 through 5. Version 3 — read the upper limit from input().
  3. Challenge Modify the seating chart to print only even-numbered rows. Show both approaches: an if statement inside the loop, and a range() step value.
chr() and ord(): ord("A") gives the integer 65. Adding a column index (0, 1, 2…) and passing the result to chr() gives the corresponding letter: chr(65) → "A", chr(66) → "B", and so on.

# ============================================================
# COP 1047C - Lecture 5 Breakout 2
# Nested Loops & Incremental Development
# ============================================================

# ----------------------------------------------------------
# PROBLEM 1: Seating chart with nested loops
# Format: row number + column letter (1A, 1B, 2A, ...)
# ----------------------------------------------------------

print("--- Problem 1: Seating Chart ---")

num_rows = int(input("Enter number of rows: "))
num_cols = int(input("Enter number of cols: "))

for row in range(1, num_rows + 1):          # outer: row numbers 1, 2, 3...
    for col in range(num_cols):              # inner: column index 0, 1, 2...
        letter = chr(ord("A") + col)         # 0->"A", 1->"B", 2->"C"...
        print(f"{row}{letter}", end="  ")    # print seat label
    print()                                  # newline at end of each row

# Sample output (input: 3 rows, 4 cols):
# 1A  1B  1C  1D
# 2A  2B  2C  2D
# 3A  3B  3C  3D

# PROBLEM 2: Incremental sum program — 3 versions
# Build a little, test a little!

# Version 1: Verify the loop iterates correctly
print("\n--- Problem 2a: Version 1 (loop check) ---")
for i in range(1, 6):
    print(i, end=" ")
print()
# Output: 1 2 3 4 5

# Version 2: Add an accumulator to sum the values
print("\n--- Problem 2b: Version 2 (sum 1-5) ---")
total = 0
for i in range(1, 6):
    total += i
print(f"Sum: {total}")
# Output: Sum: 15

# Version 3: Read the upper limit from input
print("\n--- Problem 2c: Version 3 (user-defined limit) ---")
limit = int(input("Enter upper limit: "))
total = 0
for i in range(1, limit + 1):
    total += i
print(f"Sum 1 to {limit}: {total}")

# Sample interaction (input: 10):

# ----------------------------------------------------------
# CHALLENGE: Print only even-numbered rows
# Two approaches shown — pick the one that makes more sense!
# ----------------------------------------------------------

print("\n--- Challenge: Even rows only ---")

num_rows = int(input("Enter number of rows: "))
num_cols = int(input("Enter number of cols: "))

# Approach 1: if statement inside the outer loop
print("\nApproach 1 — if statement:")
for row in range(1, num_rows + 1):
    if row % 2 == 0:                         # only process even rows
        for col in range(num_cols):
            letter = chr(ord("A") + col)
            print(f"{row}{letter}", end="  ")
        print()

# Approach 2: range() with a step of 2 (starts at 2, skips odd rows entirely)
print("\nApproach 2 — range() step:")
for row in range(2, num_rows + 1, 2):        # 2, 4, 6...
    for col in range(num_cols):
        letter = chr(ord("A") + col)
        print(f"{row}{letter}", end="  ")
    print()

# Sample output (4 rows, 3 cols):
# 2A  2B  2C
# 4A  4B  4C
